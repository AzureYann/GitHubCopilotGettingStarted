# Hello World Console App

A simple Python console application that greets the user by name with input validation.

## Features

- Prompts the user to enter their name
- Validates input to ensure:
  - Name contains **only letters** (no numbers or special characters)
  - Name is between **1-15 characters** long
- Continuously asks for valid input until requirements are met
- Displays a personalized greeting

## Usage

Run the app with:

```bash
python3 hello_world.py
```

Enter your name when prompted (letters only, 1-15 characters), and the app will greet you!

## Requirements

- Python 3.6+

## Setup

```bash
python3 -m venv venv
source venv/bin/activate
```
